This part 1 of the project consists of two main components\+:
\begin{DoxyEnumerate}
\item {\bfseries{Generator}}\+: A command-\/line tool to generate 3D primitive shapes (plane, box, cone, sphere) and save them in {\ttfamily .3d} files.
\item {\bfseries{Engine}}\+: A 3D rendering engine that reads {\ttfamily .3d} files and renders them using Open\+GL.
\end{DoxyEnumerate}

The generator program creates 3D primitive shapes and saves them in .3d files. Program Features The interactive menu provides the following options\+:


\begin{DoxyEnumerate}
\item {\bfseries{Clean and Build}}
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Cleans the build directory
\item Rebuilds both generator and engine
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item {\bfseries{Generate Figure}}
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Plane (parameters\+: unit size, slices)
\item Box (parameters\+: unit size, slices)
\item Sphere (parameters\+: radius, slices, stacks)
\item Cone (parameters\+: radius, height, slices, stacks)
\end{DoxyItemize}

All generated figures are automatically saved in tests


\begin{DoxyEnumerate}
\item {\bfseries{View Figure}}
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Lists all available generated figures
\item Renders selected figure using Open\+GL
\item Config files are stored in configs
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item {\bfseries{Exit}}
\end{DoxyEnumerate}
\begin{DoxyItemize}
\item Exits the program
\end{DoxyItemize}

Notes\+:


\begin{DoxyItemize}
\item Generated .3d files are saved in the tests directory
\item XML configuration files are stored in configs folder
\item Creating a new figure will override any existing file with the same name
